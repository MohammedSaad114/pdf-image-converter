# Converter
Simple Windows App to Convert PDF to Image or Any Image(s) to PDF

![alt text](https://github.com/baselsaad/Converter--PDF--Images-/blob/main/Screenshot.png)

# Dependencies:
 - ImageMagick
 - Ghostscript

# Build

* Open the Visual studio solution (Converter.sln) using Visual studio 2022 
* Run the project using Debug or Release config
